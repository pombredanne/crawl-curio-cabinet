Parsing Links isn't always straightforward

Base Tag

The <base> tag allows the page to specify another url to use as the base of relative urls on the page, rather than the url of the page itself

Canonical Tag

The <canonical> tag specifies the canonical url for the page that's been visited

Schemes

The schemes of links aren't limited to just http, https, etc. -- they can be things like 'mailto', 'tel', etc.

Data URI

You can actually encode data (like images) into so-called data URI's and they can be quite large in practice.
